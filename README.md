# Stocks and bonds dashboard demo

Stocks and bonds dashboard interface demo using Vue.

## Requirements
- Node 17.4.0

## Setup
Install npm dependencies
```
npm install
```

Set the .env file or the required environment variables (optional).

## Build assets
```
npm run build
```

## Start the server
```
npm run start
```

-------------------

## Dev environment
Webpack listener:
```
npm run watch
```

Server in dev mode:
```
npm run dev
```
